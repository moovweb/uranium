Toggler

Description

A toggler is an interaction that coordinates two components - a button, and a set of contents.

When you click the button, the states of both the button and the contents are toggled.

Attributes

Basic
Advanced
data-ur-set="toggler"
add this attribute to a div that wraps the whole interaction
data-ur-toggler-component="button"
add this attribute to the button
data-ur-toggler-component="content"
add this attribute to the content

data-ur-set=

"toggler"

data-ur-toggler-component=

Required

    "button"
    • add this attribute to the button (i.e. the thing to be clicked)
    • multiple?: false
    • state: enabled / disabled(default)
    "content"
    • add this attribute to each content item
    • multiple?: true
    • state: enabled / disabled(default)

Demonstration

Example

The Code

Click on the button
And content appears
<div data-ur-set="toggler">
  <div data-ur-toggler-component="button">
    Click on the button
  </div>
  <div data-ur-toggler-component="content">
    And content appears</div>
</div>

Recommended Styles 

/* Recommended SCSS Styling */
[data-ur-set='toggler'] {
  /* By Default content is hidden*/
  [data-ur-toggler-component='content'] {
    display:none;
  }
  [data-ur-toggler-component='content'][data-ur-state='enabled'] {
    display: block;
  }
}
/* Recommended CSS Styling */

[data-ur-set='toggler'] [data-ur-toggler-component='content'] {
  display:none;
}
[data-ur-set='toggler'] [data-ur-toggler-component='content'][data-ur-state='enabled'] {
  display: block;
}

Toggler Examples

Popup

If we want something to start out visible, and then disappear when we click on the button, we can just add the data-ur-state="enabled" attribute to the button and its content. 

<div data-ur-set="toggler">
  <div data-ur-toggler-component='button' data-ur-state="enabled" class='popup_button'>
    Toggle This Picture!
  </div>

  <div data-ur-toggler-component='content' data-ur-state="enabled" class='popup_content'>
     <img src='/images/popup.jpeg'>
  </div>
</div>
Toggle This Picture!

Nested Togglers

It's super-easy to nest togglers as well:

<div data-ur-toggler-component='button'>
  Foods
</div>
<ul data-ur-toggler-component='content'>
  <li data-ur-set="toggler">
    <span data-ur-toggler-component='button'> Veggies </span>
    <ul data-ur-toggler-component='content'>
      <li> Beets </li>
      <li> Carrots </li>
    </ul>
  </li>
  <li data-ur-set="toggler">
    <span data-ur-toggler-component='button'> Meats </span>
    <ul data-ur-toggler-component='content'>
      <li> Beef </li>
      <li> Fish </li>
    </ul>
  </li>
</ul>

Click on the button, and then click on the text that shows up below. (Notice how the sub-togglers even remember what state they were in when you toggle the main button.)
