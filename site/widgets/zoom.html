Zoom

Description

A widget that zooms into an image and displays a higher resolution image that can be panned.

Attributes

data-ur-set="zoom"
add this attribute to the element that wraps the whole widget
data-ur-zoom-component="view_container"
for wrapping the whole view container
data-ur-zoom-component="img"
the img element
data-ur-zoom-component="button"
zoom in/out button
data-ur-zoom-component="loading"
loading indicator when loading the larger image

data-ur-set=

"zoom"

data-ur-zoom-component=

Required

    "view_container"
    • this div should wrap the whole widget
    • multiple?: false
    • state: active / inactive (reflects currently visible item)
    • attributes
      • optional: data-ur-touch: enabled/disabled (allow clicking image to zoom)
    "img"
    • this attribute is for the image to be zoomed
    • multiple?: false
    • state: N/A
    • attributes
      • required: data-ur-src: source of the larger image
      • optional: data-ur-width: integer (width of larger image), use this to start the zoom animation before loading the larger image
      • optional: data-ur-height: integer (height of larger image), used with data-ur-width

Optional

    "button"
    • zoom in/out the image
    • multiple?: false
    • state: enabled / disabled
    "loading"
    • indicates when big image is loading
    • multiple?: false
    • state: enabled / disabled

Instance

Access the widget via: Ur.Widgets.zoom["ID"]. The following methods are exposed for each widget:
  • zoom(): zooms in/out
  • reset(): resets the zoom widget

Required Styles 

/* Required SCSS Styling */

[data-ur-zoom-component="view_container"] {
  overflow: hidden;
}

[data-ur-set="zoom"] {
  &[data-ur-state="enabled-in"], &[data-ur-state="enabled-out"] {
    [data-ur-zoom-component="img"] {
      @include transition-duration(0.4s);
      @include transition-timing-function(ease-in-out);
      -webkit-transition-property: -webkit-transform;
      -moz-transition-property: -moz-transform;
      -o-transition-property: -o-transform;
    }
  }
}

[data-ur-zoom-component="loading"][data-ur-state="disabled"] {
  display: none;
}
