--- layout: interactions title: Zoom name: zoom interaction_selected: selected description:

An interaction that zooms into an image and displays a higher resolution image that can be panned.

basic_tab: data-ur-set="zoom": add this attribute to the element that wraps the whole interaction data-ur-zoom-component="view_container": for wrapping the whole view container data-ur-zoom-component="img": the img element data-ur-zoom-component="button": zoom in/out button data-ur-zoom-component="loading": loading indicator when loading the larger image advanced_tab: Required: view_container: - this div should wrap the whole interaction - "multiple?: false" - "state: active / inactive (reflects currently visible item)" - attributes: - "optional: data-ur-touch: enabled/disabled (allow clicking image to zoom)" img: - this attribute is for the image to be zoomed - "multiple?: false" - "state: N/A" - attributes: - "required: data-ur-src: source of the larger image" - "optional: data-ur-width: integer (width of larger image), use this to start the zoom animation before loading the larger image" - "optional: data-ur-height: integer (height of larger image), used with data-ur-width" Optional: button: - zoom in/out the image - "multiple?: false" - "state: enabled / disabled" loading: - indicates when big image is loading - "multiple?: false" - "state: enabled / disabled" instance: zoom(): zooms in/out reset(): resets the zoom interaction ---
