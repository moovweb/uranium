A toggler is an interaction that coordinates two components - a button, and a set of contents.

When you click the button, the states of both the button and the contents are toggled.

basic_tab: data-ur-set="toggler": add this attribute to a div that wraps the whole interaction data-ur-toggler-component="button": add this attribute to the button data-ur-toggler-component="content": add this attribute to the content advanced_tab: Required: button: - add this attribute to the button (i.e. the thing to be clicked) - "multiple?: false" - "state: enabled / disabled(default)" content: - add this attribute to each content item - "multiple?: true" - "state: enabled / disabled(default)" ---

Nested Togglers

It's super-easy to nest togglers as well. You'll have to use ids. Give the button component and the content component the same attribute: data-ur-id="ID".
