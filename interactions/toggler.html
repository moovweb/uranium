--- layout: interactions title: Toggler name: toggler interaction_selected: selected description: |

A toggler is an interaction that coordinates two components — a button, and a set of contents.

When you click the button, the states of both the button and the contents are toggled.

basic_tab: data-ur-set="toggler": add this attribute to an element that wraps the whole interaction data-ur-toggler-component="button": add this attribute to the button data-ur-toggler-component="content": add this attribute to the content advanced_tab: Required: button: - add this attribute to the button (i.e., the thing to be clicked) - "multiple?: false" - "state: enabled / disabled (default)" content: - add this attribute to each content item - "multiple?: true" - "state: enabled / disabled (default)" Optional: drawer: - enables sliding menus for togglers on the body element - sample implementation uses custom flexbox CSS for recommended functionality - "multiple?: false" - "state: enabled / disabled (default)" js_tab: Init: | Uranium.lib.toggler({ button: "#a", content: "#b" }); ---

Recommended Styles

SCSS
CSS
    /* Recommended SCSS Styling */
    
    [data-ur-toggler-component="content"] {
      display: none;
      &[data-ur-state="enabled"] {
        display: block;
      }
    }
    
    /* Recommended CSS Styling */
    
    [data-ur-toggler-component="content"] {
      display: none;
    }
    [data-ur-toggler-component="content"][data-ur-state="enabled"] {
      display: block;
    }

Toggler Examples

Enabled

If we want something to start out visible, and then disappear when we click on the button, we can just add the data-ur-state="enabled" attribute to the button and its content.
