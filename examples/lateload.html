Late Load

the late load widget will prevent objects from loading until requested by the user. there is only one mandatory attribute

data-ur-ll-src= or data-ur-ll-href=

however data-ur-ll-event= attribute will select the group that the late loaded element is attached to.

there are 4 standard groups that handle your events for you. the click or touch, DOMContentLoaded, load and scroll if any group bares these names its event handeling and functionality are taken care of. any other group name is saved in the Ur.Widgets.late_load.components["your_group"]

NOTE: some images will not display until the page is scrolled, if this is an issue, please add the plutonium widget "auto_scroll.js"
