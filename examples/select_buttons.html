Select Buttons

Description

This widget allows you to bind increment/decrement buttons to trigger the appropriate changes on a <select> element.

Options

Demonstration

Widget

HTML

[+] [-]

<div data-ur-set='select-buttons'>
  <select data-ur-select-buttons-component='select'>
    <option selected='true' value='-1'> -None- </option>
    <option value='SKU-XS'> XS </option>
    <option value='SKU-S'> S </option>
    <option value='SKU-M'> M </option>
    <option value='SKU-L'> L </option>
  </select>
  <span data-ur-select-buttons-component='increment'> [+] </span>
  <span data-ur-select-buttons-component='decrement' data-ur-state='disabled'> [-] </span>
</div>
