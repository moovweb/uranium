Attributes

The late load widget prevents DOM objects from loading until requested by the user.
For best results open your inspector and watch the network requests from the appropriate tab.
  • data-ur-ll-src="some_src" :: used as a replacement for src tags
  • data-ur-ll-href="some_href" :: used as a replacement for href tags
  • data-ur-ll-event="group_name" :: used to define a custom group that can be loaded later

Standard Event Groups

There are 4 standard groups that are predefined to handle common events for you, If any group uses these names its event handeling and functionality is taken care of within the late load widget. Any other group name is saved in the Ur.Widgets.late_load.components["your_group"].
  • click
  • touch
  • DOMContentLoaded
  • load
  • scroll
NOTE: some images will not display until the page is scrolled, if this is an issue, please add the plutonium widget "auto_scroll.js"
