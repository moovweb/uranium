Maps Examples

But Wait! There's more...

On its widget page, the simple map demonstration just had Google Map-style pins and a list of specified addresses. Here we'll look at some more complex examples of the maps.

Only one map can be used per page. Because of this, the examples are separated out here.

The advanced example introduces geolocation and custom pin icons. The hidden map combines two widgets to make a map that only appears on a button click. Finally, the late-load map has an extra bit of javascript to make sure it only loads when a button is clicked.
