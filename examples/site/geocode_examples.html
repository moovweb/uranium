Geolocator Examples

But Wait! There's more...

There's some more tricks you can do with the geolocator widget. As you can only have one per page, they're located individually. And remember, the geolocator may not work on some browsers.

The example we demonstrated on the widget page loaded automatically as the page did. However, we can include a button to initiate this.

Also, we don't have to use a select list for the state and country. We'll see in the second example that text inputs are fine as well.
